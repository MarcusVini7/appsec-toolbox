# appsec-toolbox

Toolkit pessoal para testes de segurança web autorizados.

Inclui scripts auxiliares para validação de ambiente, enumeração básica, fingerprint web, diretórios comuns, headers HTTP, TLS e execução controlada de ferramentas como nmap, whatweb, gobuster, ffuf, nuclei, nikto e testssl.

## Aviso

Use somente em ambientes próprios ou com autorização explícita.
